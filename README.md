# AuraEdit
AuraEdit ek lightweight aur powerful Code Editor hai jo specifically mobile aur web developers ke liye design kiya gaya hai. Yeh ek Progressive Web App (PWA) hai, jiska matlab hai ise kisi bhi device par install kiya ja sakta hai aur offline use kiya ja sakta hai.
Is project ka main focus coding ko mobile par aasaan banana hai, jiske liye isme CodeMirror engine ka use kiya gaya hai jo professional syntax highlighting provide karta hai.
